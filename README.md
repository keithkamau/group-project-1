# FeedAChild
FeedAChild is a web-based platform designed to support food and economic assistance initiatives. The project aims to connect donors, volunteers, and vulnerable communities to help reduce hunger and improve livelihoods.


## Description

FeedAChild is designed as an informative and engaging website that:

- Promotes the fight against child hunger
- Showcases food assistance and economic support programs
- Encourages community involvement through volunteering and donations
- Provides awareness on global and local hunger challenges

The platform combines a **clean user interface** with meaningful content to inspire action and support.

 ## Usage
- Open the website in your browser
- Navigate through the pages:
* Home
* About Us
* Services
* Contact Us
- Learn about the mission, vision, and programs
- Explore ways to contribute (volunteering & donations)

# Features
- Fully responsive design using Tailwind CSS
- Clean and modern UI with a warm color palette
- Navigation across multiple pages
- Impact statistics section (Meals, Countries, Support, Years)
- Call-to-action sections (Volunteer & Donate)
- Structured service offerings (Economic & Food Assistance Programs)

# Pages Overview
## Home Page (index.html)
* Hero section with mission and vision
* Call-to-action (“Get Involved”)
* Impact statistics (e.g., 10 Million meals provided)
* Volunteer and donation sections

## Services Page (services.html)
* Highlights core programs:
* Cash-Based Transfers (economic support)
* Nutritional Support (food assistance)
* Social Protection (policy-based support)
* Card-based layout with images and descriptions

## Contact Page (contact.html)
* for user inquiries and communication
## About Page (about.html)
* Provides background information about FeedAChild
* Explains mission, vision, and goals
